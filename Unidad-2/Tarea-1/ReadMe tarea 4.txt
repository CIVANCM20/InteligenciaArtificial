Este proyecto resuelve un Puzzle8 (el juego) utilizando el algoritmo de búsqueda en anchura 
(BFS). El programa toma como vacio el numero 0 y almacena los estados en una cola para garantizar 
una exploración paso a paso. y prueba varios estados para optimizar la búsqueda. Una vez encontrada
la solución, se reconstruye el camino desde el estado inicial hasta la solución y se muestra la 
secuencia de movimientos junto al tiempo que tarda en ejecutarse.

usa la clase nodo tiene un estado el cual es una matriz de 3x3 y el padre es una matriz 3x3 tiene un metodo
para recuperar el la matriz actual.
